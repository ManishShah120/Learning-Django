# How to include template within template using `include Tag`
- inclde Tag
    - Synatx:-
        - {% include temp_var_name %}
        - {% include "template_name.html" %}
        - {% include "folder/template_name.html" %}
    
    - Example:-
        - {% include topvar %}
        - {% include "topcourse.html" %}
        - {% include "fees/extrafees.html" %}
